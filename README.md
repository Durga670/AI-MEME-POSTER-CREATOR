# AI-MEME-POSTER-CREATOR
An AI-powered web application that automatically generates creative event posters and memes for college events like Hackathons, Workshops, Cultural Fests, and Farewells.  This project combines AI-driven content generation with an interactive design editor to simplify poster creation while maintaining creativity and personalization.

Steps to run the project:

Frontend (React):
cd client
npm install
npm start

Backend (Node):
cd server
npm install
node server.js
